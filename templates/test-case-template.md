ID ------- Название ------- Предуслование ------ Шаги воспроизведения -------- Ожидаемый результат ------- Фактический результат

ID - Уникальный номер тест-кейса в вашей системе (TC-LOGIN-001)
Название - Краткое, информативное описание тест-кейса
Предусловие - ***Условия, которые должны быть выполнены ДО начала тестирования***
Шаги воспроизведения - ***Пошаговая инструкция выполнения тест-кейса***
Ожидаемый результат -  Как система ДОЛЖНА себя вести после выполнения шагов
Фактический результат - Статус
